Block contract

A block exports one default decorator. It receives the authored element and may enhance it with native DOM APIs or render a Preact component.

export default async function decorate(block: HTMLElement) {
  // Read authored rows, build the experience, then update the block.
}

Runtime status

initialized→loading→loaded

If loading fails, the runtime assigns data-block-status="error" and reports the failure without preventing other independent blocks from loading.

Choosing an implementation

Use Preact

Choose a component when a block has structured UI, reusable composition, asynchronous state, or meaningful interaction.

Use vanilla TypeScript

Choose a decorator for small DOM transformations with little or no application state.
