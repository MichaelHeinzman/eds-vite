# Columns Block

## Overview

The Columns Block displays authored content in responsive columns. It preserves the authored cells and decorates each row and column for layout styling.

## Features

- Responsive column layout
- Support for text, images, and mixed authored content
- Automatic image-column detection
- Lazy loading for column images
- Row and column class decoration

## Usage

The columns block initializes when `columns` markup appears on a page. Structure content as rows containing two or more columns:

| columns |
|---------|
| Column content | Column content |
| Column content | Column content |
