# Header Block

## Overview

The Header Block renders the site header using Preact. It includes the site logo, primary navigation, and a mobile-friendly menu toggle.

## Features

- Preact-rendered header
- Site logo link
- Primary navigation links
- Mobile menu toggle
- Accessible navigation labels and expanded state

## Usage

The header block is created by the page runtime and rendered into the document header. It does not require authored table content.
