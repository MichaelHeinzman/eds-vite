# Footer Block

## Overview

The Footer Block renders the site footer using Preact. It provides a simple global footer with project attribution and a link to Edge Delivery Services information.

## Features

- Preact-rendered footer
- Consistent site footer markup
- Attribution text
- External documentation link
- Block content replacement on render

## Usage

The footer block is created by the page runtime and rendered into the document footer. It does not require authored table content.
