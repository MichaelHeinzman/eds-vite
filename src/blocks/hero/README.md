# Hero Block

## Overview

The Hero Block renders a prominent above-the-fold content area with an image and text. It optimizes the first hero image for fast loading.

## Features

- Hero image and content styling
- Eager loading for the hero image
- High fetch priority for likely LCP imagery
- Image preload insertion
- Support for authored heading and text content

## Usage

The hero block initializes when `hero` markup appears on a page. Structure the block with a hero image and text content:

| hero |
|------|
| Image and hero text |
