# Cards Block

## Overview

The Cards Block displays a list of authored cards in a responsive grid. Each card can include an image and body content.

## Features

- Responsive card layout
- List and list item accessibility roles
- Automatic image-cell detection
- Lazy loading for card images
- Support for rich authored card body content

## Usage

The cards block initializes when `cards` markup appears on a page. Structure each card as a row with an optional image cell and a content cell:

| cards |
|-------|
| Image | Card text |
| Image | Card text |
| Image | Card text |
